# Command line ChatGPT 


## Example 

![Example chatgpt-cli](demo.gif)

## Setup 
You must obtain a OpenAI auth_token from here 

https://platform.openai.com/


Create a .env file 

```sh
cp .env-example .env
```

update key on .env file 
```
AUTH_TOKEN_OPEN_AI=mykey
```

## Execute
 
```sh
go run .
```
